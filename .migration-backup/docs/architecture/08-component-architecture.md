# ASCEND — Reusable Component Architecture

## Hierarchy

```
shadcn/ui primitives (never modified)
       ↓
Shared wrappers (cross-domain, styled)
       ↓
Domain components (academics / chess / guitar / startup)
       ↓
Pages (assemble domain components + wrappers)
```

---

## Layer 1: shadcn/ui Primitives

Live in `src/components/ui/`. Generated by `npx shadcn add`. **Never modified directly** — override via Tailwind classes or wrapper components.

Installed: Button, Card, Dialog, Drawer, DropdownMenu, Form, Input, Label, Select, Sheet, Tabs, Toast, Toaster, Badge, Progress, Separator, Skeleton, Tooltip.

---

## Layer 2: Shared Wrappers

### Layout

**`AppShell`**  
Props: `none` (reads UIStore internally)  
Renders: Left sidebar + topbar + `<Outlet />`. Handles sidebar collapse animation via Framer Motion `layout`.

**`Sidebar`**  
Domain nav links with animated active indicator. Badge on notification bell.

**`TopBar`**  
Global timer display (active session) + command palette trigger (`⌘K`) + notification bell + user avatar.

### Data Display

**`DataTable<T>`**  
```typescript
interface DataTableProps<T> {
  data: T[];
  columns: ColumnDef<T>[];
  isLoading?: boolean;
  emptyState?: React.ReactNode;
  onRowClick?: (row: T) => void;
  filterKey?: keyof T;
}
```
Wraps TanStack Table with Tailwind styling. Handles loading skeleton + empty state automatically.

**`EmptyState`**  
```typescript
interface EmptyStateProps {
  icon: LucideIcon;
  title: string;
  description?: string;
  action?: { label: string; onClick: () => void };
}
```

**`HeatmapCalendar`**  
```typescript
interface HeatmapCalendarProps {
  data: { date: string; value: number }[];
  colorScale?: string[];    // Tailwind bg classes from low→high intensity
  tooltip?: (day: { date: string; value: number }) => string;
  year?: number;
}
```
GitHub-style contribution grid. Used for: habit streaks, study hours, chess sessions.

### Charts

All charts wrap Recharts. Consistent API:

```typescript
interface BaseChartProps {
  data: { label: string; value: number; [key: string]: unknown }[];
  height?: number;
  color?: string;        // Primary hex
  loading?: boolean;
  emptyMessage?: string;
}
```

- **`LineChart`** — Rating trends, score over time, daily study hours
- **`BarChart`** — Subject-wise study time, weekly comparison
- **`RadarChart`** — Domain balance (5 axes: academics, startup, chess, guitar, life)
- **`AreaChart`** — Cumulative study hours, streak history

### Forms

**`FormField`** — Wraps react-hook-form `Controller` + shadcn Label + shadcn Input + error message in one component.

```typescript
interface FormFieldProps {
  name: string;
  label: string;
  control: Control<any>;
  type?: 'text' | 'number' | 'date' | 'textarea' | 'select';
  options?: { label: string; value: string }[];  // for select
  placeholder?: string;
  description?: string;
}
```

**`DatePicker`** — Wraps shadcn Calendar inside a Popover. Returns `Date | undefined`.

**`TagInput`** — Comma-separated tag entry with chip display. Controlled.

### Goals & Tasks

**`GoalCard`**  
Shows: title, domain badge, status, progress bar (animated via Framer Motion), target date.  
Click → opens `GoalDetailSheet`.

**`GoalProgress`**  
Animated circular progress ring (SVG) + percentage. Used in dashboard domain summary cards.

**`TaskCard`**  
Draggable (via `@dnd-kit/core`) for Kanban view. Shows: title, priority dot, due date, domain chip.

**`TaskKanban`**  
Four-column board: Todo | In Progress | Done | Cancelled. Uses `@dnd-kit` drag-and-drop.

### Habits

**`HabitRow`**  
One row in the habits list. Checkbox (logs today's check-in on click). Shows streak count + mini spark-line of last 7 days.

**`HabitStreak`**  
Displays `🔥 N days` with colour based on streak length. Animated on increment.

### Timer

**`SessionTimer`**  
Floating bottom-right panel when a session is active. Shows: domain icon + entity name + elapsed time + pause/stop buttons. Persists across page navigation via portal.

```typescript
interface SessionTimerProps {
  domain: Domain;
  entityLabel: string;    // e.g. "Chapter 3: Thermodynamics"
  mode: 'pomodoro' | 'free';
}
```

### AI

**`AIChat`**  
Slide-in panel (shadcn Sheet). Renders conversation history + streaming response. Handles SSE from Edge Function.

**`AISuggestion`**  
Inline dismissable card. Shows AI-generated suggestion with "Use this" action.

## Animation Conventions (Framer Motion)

```typescript
// Standard page enter
export const pageVariants = {
  initial: { opacity: 0, y: 12 },
  animate: { opacity: 1, y: 0, transition: { duration: 0.25, ease: 'easeOut' } },
  exit:    { opacity: 0, y: -8, transition: { duration: 0.15 } },
};

// Staggered list
export const listVariants = {
  animate: { transition: { staggerChildren: 0.06 } },
};

export const listItemVariants = {
  initial: { opacity: 0, x: -8 },
  animate: { opacity: 1, x: 0 },
};

// Habit check-in bounce
export const checkInVariants = {
  checked:   { scale: [1, 1.3, 1], transition: { duration: 0.3 } },
  unchecked: { scale: 1 },
};
```

All layout changes (sidebar open/close, card expand) use `layout` prop for automatic Framer Motion layout animations.

## Component Naming Rules

- `XxxPage` — Route-level component (lives in `src/pages/`)
- `XxxCard` — Read-only summary card
- `XxxRow` — Table/list row variant
- `XxxForm` — Form-only (no wrapping card/modal)
- `XxxModal` — Self-contained with Dialog
- `XxxSheet` — Slide-in panel using shadcn Sheet
- `XxxChart` — Chart wrapper

All shared components are exported from `src/components/shared/index.ts` for clean imports.
