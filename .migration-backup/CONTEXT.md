## What is ASCEND?
ASCEND is a personal self-improvement tracking app built exclusively for Prince. It tracks progress across multiple life domains to help him build the skills, habits, and achievements needed for Stanford admission.

**Motto:** Focus. Execute. Ascend.

---

## Tech Stack
- **Monorepo:** pnpm workspaces
- **Language:** TypeScript
- **Frontend:** React (Vite)
- **Backend:** Node.js/Express
- **AI:** OpenRouter API (key stored in Replit Secrets as `OPENROUTER_API_KEY`)
- **Styling:** Tailwind CSS (dark theme)

## App Sections
| Section | Purpose |
|---|---|
| Dashboard | Mission Control — daily score, streak, active goals, weekly progress, upcoming tasks |

| Profile | Prince's personal profile |
